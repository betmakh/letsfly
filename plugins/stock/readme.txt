/*
 *	Русский
 *
 *	Плагин автоматического ресайза и открытия изображения и группы изображений 
 *	в LightBox(всплывающее изображение) и в виде MobilySlider (слайдшоу, слайдер).
 *	А так же с CloudZoom эффектом  (увеличение изображения, зум эффект).
 *  	Альбомы c liPaginate, jQuery пагинация (разбивка на страницы) и "title".
 *	Фотосток+ плагин v1.3.2 для Monstra CMS
 *
 *	Установка:
 *	Загрузите архив и распакуйте его
 *	Переместите папку "stock" в папку "plugins" на вашем сервере
 *	В админ панели откройте плагины > установить, установите плагин Stock
 *	Если плагин не устанавливается проверьте права на запись к папке темы
 *
 *	Использует jQuery v1.7.2
 *	Создает стили и скрипты в папке с темой
 *	Тестировалось на Monstra CMS v2.2.0
 *
 *	Базируется на исходниках:
 *	LightBox - jQuery виджет открытия изображений в Модальном окне
 *	MobilySlider - jQuery виджет CлайдШоу изображений
 *	CloudZoom - jQuery виджет увеличения изображений
 *	liPaginate  - jQuery виджет разбивки на страницы
 *	Плагин Фотосток v1.0.0 для Monstra CMS
 *
 *	Вы сможете найти подробную инструкцию после установки плагина в административной панели на странице Фотостока+.
 */
 
/* 
 *	Параметры LightBox.
 *
 * 	Если плагин установлен можно установить Лайтбокс и изображениям вне Фотостока+. Добавляя атрибут rel="lightbox"  
 *	ссылке на изображение и  rel="lightbox[albumx]" каждой ссылке при создании группы изображений. 
 *	Так-же при таком выводе можно добавить title="name-text" который отобразится на всплывающем изображении.
 */
 
/* 
 *	Параметры MobilySlider. Добавлять и редактировать в int-mobilyslider-stock.js. 
 *	Для каждого слайдера в зависимости от альбома по порядковому номеру.
 * 
 *	content: '.sliderContent', // селектор для слайдера
 *	children: 'div', // селектор для дочерних элементов
 *	transition: 'horizontal', // переходы: horizontal, vertical, fade
 *	animationSpeed: 300, // скорость перехода в миллисекундах
 *	autoplay: false,
 *	autoplaySpeed: 3000, // время между переходами (миллисекунды)
 *	pauseOnHover: false, // останавливать навигацию при наведении на слайдер: false, true
 *	bullets: true, // генерировать навигацию (true/false, class: sliderBullets)
 *	arrows: true, // генерировать стрелки вперед и назад (true/false, class: sliderArrows)
 *	arrowsHide: true, // показывать стрелки только при наведении
 *	prev: 'prev', // название класса для кнопки назад
 *	next: 'next', // название класса для кнопки вперед
 *	animationStart: function(){}, // вызывать функцию при старте перехода
 *	animationComplete: function(){} // вызывать функцию когда переход завершен
 *
 *	Если плагин установлен можно установить MobilySlider и изображениям вне Фотостока+.     
 *	<div class="slider outsidestock">
 *       <div style="width:800px; height:200px;" class="sliderContent">
 *            <div class="item">
 *               <img src="img/img1.jpg" alt="" />
 *            </div>
 *            <div class="item">
 *                <img src="img/img2.jpg" alt="" />
 *            </div>
 *        </div>
 *	</div>
 *	Незабывая добавить инициализацию внешнего слайдера в int-mobilyslider-stock.js. 
 *	А именно (по примеру выше) инициализацию "outsidestock", аналогично инициализации slide-1, slide-2 и т.д.
 */
 
/*  
 *	Параметры CloudZoom. Редактировать в zoom-stock.js. В конце скрипта zoom-stock.js, будут видны установленные параметры.
 *
 *	zoomWidth	ширина zoom-окна, при значении "auto" будет равна ширине окна с маленькой картинкой	
 *	zoomHeight	высота zoom-окна, при значении "auto" будет равна высоте окна с маленькой картинкой	
 *	position	определяет позицию zoom-окна по отношению к окну-родителю, может принимать значения "left", "right", "top" и "bottom", 
 *	а также "blockId" — id блока, в случае, если нужно разместить zoom-окно в другой части экрана	
 *	adjustX	корректировка положения zoom-окна по горизонтали
 *	adjustY	корректировка положения zoom-окна по вертикали
 *	tint	цвет фона вокруг рамки родительского окна в hex-формате(#RRGGBB) (Не работает в паре с softFocus)
 *	tintOpacity	прозрачность фона вокруг рамки-курсора в родительском окне
 *	lensOpacity	прозрачность рамки-курсора в родительском окне
 *	softFocus	эффект размытия области вокруг рамки-курсора (не работает в паре с tint)
 *	smoothMove	сглаживание движения изображения в zoom-окне(значение 1 будет в точности повторять движение мыши по родительскому окну)
 *	showTitle	показывать описание (берется из атрибута title изображения)
 *	titleOpacity	прозрачность фона с описанием
 * 
 *	Если плагин установлен можно установить CloudZoom и изображениям вне Фотостока+. 
 *	Добавляя атрибуты class="cloud-zoom" и пустой rel=""  ссылке на изображение, внутри которой находится привью изображения.
 */ 
 
 
 
/*  
 * 	Параметры liPaginate. Редактировать в int-paginate-stock.js.
 *	
 * 	jQuery Auto Pagination или автоматическая пагинация текста
 * 	Размеры страниц зависят от заданного параметра высоты, который можно указывать в пикселях и присвоить ему значение "auto". 
 * 	При указании в качестве высоты значения "auto", размер страниц будет ориентироваться на размер экрана.
 * 	Также плагин liPaginate сам добавит навигацию по созданным страницам. Положение навигации также можно указать в параметрах плагина: 
 * 	'before' - перед страничками, 'after' - после страничек и 'double' - одновременно и перед, и после страниц. 
 *
 * 	В плагине 3 эффекта переключения страниц:
 * 	simple - стандартный без эффектов, 
 * 	fade - плавное затухание страничек, 
 * 	hSlide - горизонтальный слайд страничек, 
 * 	vSlide - вертикальный слайд страничек
 */
